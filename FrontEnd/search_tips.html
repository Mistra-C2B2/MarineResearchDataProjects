The C2B2-relevant list of research projects

Search tips

You can search the list using your browser's search function, usually activated with Ctrl-F:

  • If you only want to search for words in the project's titles, first Collapse all the descriptions using the button at the top of the page.
  • If you want to search for words in the projects' abstracts, first use the Expand button at the top of the page.
  • You can search in either English of Swedish by toggling the language using the EN SE buttons at the top of the page.
